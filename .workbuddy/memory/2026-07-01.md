# 2026-07-01 工作日志

## 今日工作

1. 阅读了 EG2226 栅极驱动芯片规格书，提取关键参数：VCC 10~20V、高端 600V 悬浮、500kHz 开关、0.6A/1.0A 驱动能力、输入高电平阈值 2.5V、自举电路结构。
2. 详细分析了用户提供的 122火箭弹_四合一舵机驱动板 原理图（3 页），发现并记录多处设计问题：
   - VDD_5V 分压电阻接反导致 MCU ADC 输入 4.55V，会损坏 PA0。
   - VDD_VM 分压比 1:1，VM>6.6V 时 PA1 过压损坏。
   - INA139 输出无钳位，大电流时 ISEN_VM 可能超过 3.3V 损坏 PA6。
   - TPS54302 输入耐压 28V 与 VM 可能达到 35V 不匹配。
   - EG2226 自举电容 0.1uF 偏小，建议加大到 1uF。
   - DSK34 自举二极管耐压需确认是否满足 24V 以上 VM。
   - 0.005Ω 采样电阻 1W 功率限制对应约 14A 连续电流，需确认电机电流。
   - 缺少栅极串联电阻、电机反电动势保护、RS485 终端电阻等。
3. 将 PDF 原理图渲染为 PNG 以便逐页视觉检查。
