## 频率架构约束（控制理论）

- 当前：主中断 TIM3=16k，速度环=16k/4=4k，位置环=16k/16=1k，PWM(motor_freq)=10k。
- 硬约束：主中断 ≥ 速度采集 ≥ 速度环 ≥ 位置环；内环(速度) ≥ 3~10× 外环(位置)，当前 4× 合格。
- 带宽上限：速度环可用带宽 ≤ 4k/10 = 400Hz；位置环 ≤ 1k/10 = 100Hz。
- 风险点：速度滤波 speed_avg(5点)+speed_lpf，若速度在 4k 才更新则延迟≈0.5~1.25ms，接近/超过速度环周期 0.25ms，会压低 ki/kd 上限。建议速度采集放到 16k（每中断算），PID 仍 4k。
- 注意：My_Tim_Callback 中速度更新 `Update_Speed_Angle_LPFAndPLL()` 仍注释，闭环未跑。
- 实际瓶颈：4 路 KTH7111 共享 SPI3，速度环 4k → 16k 次/秒 SPI 读，需核算 SPI3 波特率能否在 250µs 内读完 4 路+算 PID。

## 推荐频率配置（用户确认速度采集=16k）

- 主中断 TIM3 = 16k（采样节拍，保持）
- 速度采集 = 16k（每中断算速度/角度）
- 速度环 = 4k（div=4，4×位置环，带宽≤400Hz，甜点；不够再升 8k/div=2）
- 位置环 = 1k（div=16，带宽≤100Hz，保持）
- PWM 输出 = 20k（实践甜点：静音+低纹波；当前默认10k会啸叫；EG2222支持到500k无压力）
- 电流环 = 暂不设（仅INA139监测）；若加则≥40k且ADC与PWM中心对齐同步
- 说明：PWM频率与控制环独立，PWM只管开关纹波/噪声，无需与控制环整数倍。
